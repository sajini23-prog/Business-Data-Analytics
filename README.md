##### Business Data Analytics Project

##### Overview

###### This project analyses online retail sales data using Python, SQL, and data visualisation.

##### It includes:

###### Creating a database using SQLite
###### Cleaning and storing data from CSV files
###### Running SQL queries to analyse sales and customers
###### Visualising results using charts

##### Tools Used

###### Python
###### Pandas
###### SQLite
###### Matplotlib
###### Seaborn

##### How to Run

###### 1. Install required libraries
###### 2. Open and run `Main.ipynb`
